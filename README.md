The university has user and an admin interface.
The User interface is for the users(students, parents etc).
The admin interface is for the admin/staff - here they can see the applications that have been submitted.
The User interface has pages such as home, about, academics, admissions, news, and contact page and also an online application form.
The database file is included and is named university.sql
There is a live site.
